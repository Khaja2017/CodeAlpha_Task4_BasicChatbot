# Basic Chatbot - Task 4

This is a simple rule-based chatbot created using Python.

## Features
- Responds to simple messages like "hello", "how are you", and "bye"
- Uses conditional statements to generate responses
- Runs continuously until the user types "bye"

## Technologies Used
- Python
- if-elif conditions
- Functions
- Loops
- User input/output

## How to Run

Run the program using:

python chatbot.py
